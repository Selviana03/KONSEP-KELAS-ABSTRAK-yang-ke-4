1. Abstract Class & Abstract Method

Abstract Class adalah kelas yang tidak bisa dibuat objeknya secara langsung dan berfungsi sebagai kontrak. Abstract Method adalah metode tanpa isi yang wajib diimplementasikan oleh kelas turunan.

abstract class Pekerja {
  void bekerja(); // Abstract method
}

2. Implementasi Pewarisan (Inheritance)

Kelas turunan seperti Programmer, Dokter, dan Guru mewarisi kelas Pekerja dan memberikan isi spesifik pada metode bekerja() menggunakan @override.

3. Polimorfisme (Polymorphism)

Kemampuan objek untuk memiliki banyak bentuk. Kita bisa menyimpan objek Programmer dan Dokter ke dalam sebuah List<Pekerja> dan menjalankan metode yang sama namun dengan hasil berbeda.

List<Pekerja> daftar = [Programmer('Serlin'), Dokter('Siti')];

4. Interface (Multiple Inheritance)

Di Dart, kita menggunakan kata kunci implements untuk meniru multiple inheritance. Sebuah kelas Tablet bisa memiliki kemampuan dari kelas Komputer, Kamera, dan Telepon sekaligus.

5. Static Methods (MathUtils)

Metode Static memungkinkan kita mengakses fungsi di dalam kelas tanpa perlu membuat objek (instansiasi). Digunakan dalam kelas MathUtils untuk fungsi matematika seperti faktorial, cek prima, dan rata-rata.

class MathUtils {
  static int factorial(int n) => ... ;
}
// Cara panggil: MathUtils.factorial(5);
